# Website Image Size Guide

This guide provides the recommended dimensions and aspect ratios for images across the website. Providing images at these specifications will ensure the best quality and performance.

> **Note to Designer:**
> *   All images should be optimized for web (JPEG for photos, PNG/SVG for graphics).
> *   **Safe Zones:** For Hero images, keep important content (faces, text) in the **center**, as edges may be cropped on different screen sizes (mobile vs. desktop).
> *   **Resolution:** Dimensions below are for standard displays. For high-DPI (Retina) screens, you can provide images at **2x** these dimensions.

---

## 1. Home Page / Landing (`/`)

| Section | Image Type | Recommended Size | Aspect Ratio | Notes |
| :--- | :--- | :--- | :--- | :--- |
| **Hero (Automation)** | Background | **1920 x 1080 px** | 16:9 | Full-screen background. Content may be cropped on mobile. |
| **Hero (Manufacturing)** | Background | **1920 x 1080 px** | 16:9 | Full-screen background. |
| **About Section** | Feature Image | **800 x 600 px** | 4:3 | Appears on the right side of the "About Interlink" section. |
| **Services Section** | Card Images | **600 x 400 px** | 3:2 | Images for the 3 featured services cards. |

## 2. Products Page (`/products`)

| Section | Image Type | Recommended Size | Aspect Ratio | Notes |
| :--- | :--- | :--- | :--- | :--- |
| **Hero Section** | Background | **1920 x 600 px** | ~3:1 | *Currently uses CSS gradients*, but if an image is added, use this size. |
| **Product Grid** | Product Cards | **600 x 400 px** | 3:2 | Main product thumbnails. Consistent aspect ratio is key for the grid. |

## 3. Services Page (`/services`)

| Section | Image Type | Recommended Size | Aspect Ratio | Notes |
| :--- | :--- | :--- | :--- | :--- |
| **Hero Section** | Background | **1920 x 600 px** | ~3:1 | *Currently uses CSS gradients*, but if an image is added, use this size. |
| **Service Cards** | Feature Images | **800 x 500 px** | ~16:10 | Detail images for each service card. |
| **Process/Workflow** | Icons/Graphics | **SVG or 64 x 64 px** | 1:1 | Icons for "Consultation", "Engineering", etc. |

## 4. About Us Page (`/about`)

| Section | Image Type | Recommended Size | Aspect Ratio | Notes |
| :--- | :--- | :--- | :--- | :--- |
| **Hero Section** | Background | **1920 x 800 px** | ~2.4:1 | Background for "Engineering Excellence". |
| **Corporate Overview** | Office/Team | **1000 x 800 px** | 5:4 | Large feature image next to "Who We Are". |
| **CEO Message** | Portrait | **600 x 800 px** | 3:4 (Portrait) | **Important:** CEO photo. Vertical orientation works best here. |
| **Certifications** | Logos | **400 x 300 px** | 4:3 | Logos should be on a transparent background (PNG). |

---

## Summary of Common Sizes

*   **Full Screen Backgrounds:** 1920 x 1080 px
*   **Card Thumbnails (Products/Services):** 600 x 400 px
*   **Feature Images (About/Office):** 800 x 600 px
*   **Portraits (CEO):** 600 x 800 px
